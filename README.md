# Retail Sales Data Analysis (Python)

## Project Overview
This project analyzes a retail sales dataset using Python to understand customer behavior, sales trends, and product performance.

## Objective
- Understand how customers purchase products
- Compare sales by gender and age
- Analyze price vs quantity relationship
- Find useful business insights from data

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Information
The dataset contains retail transaction data with the following columns:
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## Analysis Performed
- Data loading and basic exploration
- Data cleaning (fixing gender values, date format)
- Feature engineering (month, year)
- Sales analysis by gender and product category
- Age distribution analysis
- Price vs quantity analysis
- Data visualization using charts

## Key Insights
- Electronics category generates the highest revenue
- Male and Female customers contribute almost equally to total sales
- Most customers are between 25 to 45 years old
- Lower-priced products are sold in higher quantities
- Higher-priced products are sold in lower quantities
